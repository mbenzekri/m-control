Conditional rendering

Use m-if modifier to manage condiational rendering of node subtree. the attribute value is the boolean expression to render or not the corresponding node subtree 

m-if="µ.visible" 
m-if="(µ.person.name === 'Tac')"

IMPORTANT: m-if node subtree is inserted or removed from document depending on the boolean expression not just hidden

If you want just a hidding behavior use m-set/m-bing with 'hidden' property.
