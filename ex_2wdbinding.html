input element with two way data binding

To bind a model property to an input element use the 'm-on:' modifier followed by the property you want to bind.

The demo :
My name is {{ µ.name }}
My preferred number is {{ µ.number }}
My preferred city is {{ µ.city }}
My prefered browser is {{ µ.browser }}

This example bind the model property 'name' of the encompassing Controller with the 'value' property of the input element.
